# Local-LLM-Performance-Prompt-Comparison
This project explores local LLM performance using [Ollama] with the Qwen3:1.7b model. Different prompting techniques were tested to generate AI Engineering LinkedIn posts, comparing output quality, structure, speed, and hardware performance during local AI inference.

# Local LLM Performance & Prompt Comparison
## Exploring Local Inference with Ollama

---

## Project Overview

This project demonstrates how to run a Local Large Language Model (LLM) using Ollama and compare different prompting techniques on a real-world task.

The project evaluates:

- Local model setup using Ollama
- Model performance on local hardware
- Prompt engineering effectiveness
- Response quality across prompt styles
- Advantages and limitations of local inference
